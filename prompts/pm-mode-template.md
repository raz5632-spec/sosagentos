# PM Mode Report Template

| Item | Required output |
|---|---|
| Completed | What was actually implemented |
| Files changed | Exact files added/modified/deleted |
| DB changes | Migration names, schema impact, rollback note |
| APIs changed | Routes, contracts, providers, webhooks |
| Tests | Unit/integration/e2e coverage added or updated |
| Documentation | Docs pages and task card updated |
| Risks | Known limitations, assumptions, debt |
| Blockers | Remaining blockers, if any |
| External actions required | Accounts, OAuth, App Review, DNS, secrets, approvals |
| Recommended next task | Single next-best task with rationale |
